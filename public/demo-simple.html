Google OAuth Demo (Simple)
Continue with Google
Setup Instructions:
1. Go to Google Cloud Console
2. Navigate to APIs & Services > Credentials
3. Edit your OAuth 2.0 Client ID
4. Add http://localhost:3002 to Authorized JavaScript origins
5. Add http://localhost:3002/api/v1/auth/google/callback to Authorized redirect URIs
6. Save and try again
